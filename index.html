Quetzal
A simple and lightweight text and code editor.

Warning! This is an experimental package, it's an old project of mine being rebranded and ported from Vala to C. So, in the meanwhile, you can use that project ( Simple Text ) to test the features that are planned for this one.

Quetzal is an open source text and code editor written in C using the GTK+3 toolkit, taking advantage of the GTKSourceView, VTE and others tools to provide its features. It's the rebranding of an old project of mine.

It's planned to be available as an AUR package for ArchLinux and we are working on making it available for other Linux distributions such as Debian and Fedora.


Installation

Check the releases page for distribution-based installation options.

If there isn't a quetzal package for your distribution, check the GitHub page for compiling and installation instructions.
